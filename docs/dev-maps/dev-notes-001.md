# Dev Notes 001 — Verification Guide

This document outlines how to verify the changes implemented during the development cycle on 2026-03-14.

## 1. Apple Intelligence Expansion

The `AppleIntelligenceService` has been expanded beyond title generation to include summarization, writing suggestions, and a streaming chat provider.

### Verification Steps:
1. **Chat Provider:**
   - Open WriteVibe on a device running macOS 26+ with Apple Intelligence enabled.
   - Start a new thread.
   - Click the model picker and select **Apple Intelligence**.
   - Send a message (e.g., "Hello").
   - Verify that the response streams in and is generated by the on-device model.
2. **Model Picker UI:**
   - Verify that "Apple Intelligence" no longer says "Legacy" in its subtitle or tagline.
   - Hover over it to see the updated description.

## 2. Capability Chips (Stubs)

Capability chips in the `ChatInputBar` are now interactive and reflect their current state.

### Verification Steps:
1. **Toggles:**
   - Click **Search** to toggle it on/off. Verify the background color changes to the accent color when active.
   - Click **Memory** to toggle it on/off. Verify it is active by default.
2. **Menus:**
   - Click **Tone**, **Length**, or **Format**.
   - Select a different option from the menu (e.g., Change Tone from "Balanced" to "Professional").
   - Verify the chip label updates and the chip becomes active (accent color background).

## 3. Sidebar Apps & Library (Stubs)

The sidebar items are now interactive buttons that provide feedback.

### Verification Steps:
1. **Apps Section:**
   - Expand the **Apps** section in the sidebar.
   - Click **Images**, **Canvas**, **Templates**, or **Mood Board**.
   - Verify a toast notification appears at the bottom saying "X app coming soon".
2. **Library Section:**
   - Expand the **Library** section.
   - Click **Emails**, **Stories**, **Essays**, etc.
   - Verify a toast notification appears saying "X library coming soon".
   - Note: **Articles** should still navigate to the Articles dashboard.

## 4. Automated Tests

New unit tests have been added for core services.

### Verification Steps:
1. **Run Tests:**
   - Open `WriteVibe.xcodeproj` in Xcode.
   - Press `Cmd + U` or go to **Product > Test**.
   - Verify that `StreamingServiceTests` and `ConversationServiceTests` pass.
   - Alternatively, run from CLI:
     ```bash
     xcodebuild test -project WriteVibe.xcodeproj -scheme WriteVibe -destination 'platform=macOS'
     ```

## 5. Regression Check

1. **Ollama / Cloud Models:**
   - Ensure you can still switch back to Ollama or a Cloud provider (Anthropic/OpenRouter).
   - Verify streaming still works for these providers.
2. **Article Editor:**
   - Navigate to **Library > Articles**.
   - Open an article and ensure the editor still loads correctly.
